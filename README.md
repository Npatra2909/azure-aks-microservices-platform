# Azure AKS Microservices Platform

End-to-end DevOps project using:
- Azure (AKS, ACR, Monitor)
- Terraform (IaC)
- GitLab CI (CI/CD)
- Kubernetes (microservices)
- Observability (Prometheus, Grafana, Logs)

## Structure

- apps/ → microservices
- infra/ → Terraform infrastructure
- k8s/ → Kubernetes manifests
- docs/ → architecture and flows
